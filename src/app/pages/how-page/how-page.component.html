How do we know?

We use the Mitre Attack group list (available here). We then link each group to a set of activity area which it is known to target. This association is done manually, by studying the description of those groups and their known attacks.

We are then able to filter our group list based on those activity area tags we gave them.

Why did we do this?

This web app is part of a scholarship assignment in our Threat Intelligence course. We had a limited time to build a solution linked to Threat Intelligence. This idea came as we asked ourselves the question of which group we might be able to encounter in our respective companies.
